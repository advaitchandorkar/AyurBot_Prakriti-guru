Local setup
1) Backend
- Create `.env` from `.env.example`
- `python -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`
- `python main.py`

2) Frontend
- `cd frontend`
- `npm install`
- `npm run build`

3) Open http://localhost:5001
